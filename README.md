## How to Play
1. Install a cobol compiler  
`sudo apt install open-cobol`  
2. Compile the COBOL file  
`cobc -W -x -o TicTacTOBOL TicTacTOBOL.cbl`  
3. Run the executable  
`./TicTacTOBOL`  
4. Play the game
